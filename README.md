STRAT

If there is a draw stack:
- play Wild Draw first if possible
- otherwise play a coloured Draw card
- choose the Draw card that keeps your strongest colour afterwards

If there is no draw stack:
- find all legal cards
- prefer non-wild cards first
- play high-value cards first to reduce final score
- use colour-counting as a tiebreaker
- only use Wild/Wild Draw when no normal card is playable
- when using a Wild, choose the colour you have most remaining
